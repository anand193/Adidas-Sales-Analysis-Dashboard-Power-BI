# 👟 Adidas Sales Analysis Dashboard – Power BI

## 📌 Project Overview
This project analyzes Adidas sales data to uncover key business insights related to revenue, profit, regional performance, and product trends.  
The dashboard helps stakeholders monitor KPIs, identify high-performing regions, and optimize sales strategy using data-driven insights.

---

## 🎯 Business Problem
Adidas wants to:
- Track overall sales performance
- Identify top-performing regions and retailers
- Analyze product category profitability
- Monitor monthly sales trends
- Improve decision-making using interactive dashboards

---

## 📊 Key Performance Indicators (KPIs)
- 💰 Total Sales
- 📈 Total Profit
- 📦 Total Units Sold
- 📊 Profit Margin %
- 🏬 Sales by Region
- 🛍 Sales by Product Category
- 📅 Monthly Sales Trend

---

## 🗂 Dataset Description
The dataset contains:
- Invoice Date
- Retailer
- Region
- State
- Product Category
- Units Sold
- Sales Revenue
- Operating Profit
- Price per Unit

Data was cleaned and transformed using Power Query before visualization.

---

## 🛠 Tools & Technologies Used
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Excel (for initial data handling)

---

## 📈 Dashboard Features
- Interactive filters (Region, Invoice Date)
- Dynamic KPI cards
- Trend analysis using line charts
- Region-wise performance map
- Category-wise profit comparison

---

## 🔍 Key Insights
- The West region generated the highest revenue.
- Certain product categories contributed disproportionately to total profit.
- Seasonal trends indicate peak sales during specific months.
- A small number of retailers contributed to a large percentage of revenue (Pareto insight).
